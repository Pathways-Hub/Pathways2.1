JavaScript Plugin Builder

This documentation shows how to build custom JavaScript tools that plug into this app. All elements use your existing style classes like control-btn, extra-container, and import-description.

START HERE: Plugin Function

Begin by writing a function to define your plugin. Replace MyPlugin with your actual tool name.

function MyPlugin() {
    console.log("Plugin started");
}

ADD A BUTTON

Create a button to launch your plugin. It will use the class control-btn to match your existing UI.

const btn = document.createElement("button");
btn.className = "control-btn";
btn.textContent = "Launch MyPlugin";
btn.onclick = MyPlugin;
document.body.appendChild(btn);

CREATE A CONTAINER

Use this to spawn a styled box for your tool. Containers can hold text, inputs, and buttons.

const container = document.createElement("div");
container.className = "extra-container";
document.body.appendChild(container);

INSERT TEXT

Add paragraph-style text with the import-description class for visual consistency.

const text = document.createElement("p");
text.className = "import-description";
text.textContent = "Hello from MyPlugin";
container.appendChild(text);

INPUT FIELD

You can add an input box to collect user data.

const input = document.createElement("input");
input.placeholder = "Type here...";
container.appendChild(input);

const value = input.value; // use this to get input

ADD A BUTTON TO THE TOOL

Add an internal plugin button to perform actions, like saving or submitting.

const innerBtn = document.createElement("button");
innerBtn.className = "control-btn";
innerBtn.textContent = "Submit";
innerBtn.onclick = () => alert(input.value);
container.appendChild(innerBtn);

MAKE IT DRAGGABLE

Add drag support to your plugin container so users can move it around.

function makeDraggable(element) {
    let offsetX, offsetY, isDragging = false;

    element.addEventListener("mousedown", (e) => {
        if (e.target.tagName === "INPUT" || e.target.isContentEditable) return;
        isDragging = true;
        offsetX = e.clientX - element.offsetLeft;
        offsetY = e.clientY - element.offsetTop;
        element.style.zIndex = 9999;
    });

    document.addEventListener("mousemove", (e) => {
        if (!isDragging) return;
        element.style.left = `${e.clientX - offsetX}px`;
        element.style.top = `${e.clientY - offsetY}px`;
    });

    document.addEventListener("mouseup", () => {
        isDragging = false;
    });
}

FULL PLUGIN EXAMPLE

Here's the complete pattern to spawn a draggable plugin with input, button, and text.

function MyPlugin() {
    const container = document.createElement("div");
    container.className = "extra-container";
    container.style.position = "absolute";
    container.style.top = "300px";
    container.style.left = "300px";

    const text = document.createElement("p");
    text.className = "import-description";
    text.textContent = "This is MyPlugin tool";
    container.appendChild(text);

    const input = document.createElement("input");
    input.placeholder = "Your input...";
    container.appendChild(input);

    const btn = document.createElement("button");
    btn.className = "control-btn";
    btn.textContent = "Submit";
    btn.onclick = () => alert(input.value);
    container.appendChild(btn);

    document.body.appendChild(container);
    makeDraggable(container);
}

IMPORTANT PLUGIN CONSTRAINTS

To ensure performance and compatibility, all plugins must follow these rules:

  • Max file size: Your entire plugin must be under 500KB.
  • Single file only: Plugins must be contained in a single JavaScript file.
  • No local assets: Any images or media must use hosted URLs (e.g., https://example.com/image.png). Local file paths like ./img/pic.png will not work.

Stick to the structure provided in this guide to ensure your plugin is accepted and functions correctly inside the workspace.
